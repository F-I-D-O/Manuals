# Android Debugging Bridge (ADB)
ADB is a command-line tool that allows you that allows to manipulate Android devices from the PC command line. To use ADB, we need to:

1. Download the Android [Platform Tools](https://developer.android.com/tools/releases/platform-tools) that contains the ADB tool and extract it to a folder.
1. On the Android device, enable the "Developer options" 
1. Connect the Android device to the PC via USB and select "Transfer files" mode.

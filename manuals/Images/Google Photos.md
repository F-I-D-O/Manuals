# Adjusting Timezone
Sometimes, a wrong timezone is set for the photos. This happens when the source device has a wrong timezone set, or does not set the timezone at all. To fix this:

1. Clean and reupload the photos in the album as necessary so that we can select only the photos with the wrong timezone.
2. Select all photos with the wrong timezone.
3. Click on the three dots in the top right corner then `Edit date & time` -> `Shift dates & times`.
4. Look at the time range of the photos in the bottom of the dialog.
5. Set the timezone to the correct one.
6. The time is now shifted with the time zone. Set the time manually so that the time range is the same as before.

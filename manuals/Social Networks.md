# Instagram

## Posts
Posts can be added both from computer and from mobile device. However, there are differences:

| Feature | Mobile | Web |
| - | - | - |
| Image limit | 20 images per post | 10 images per post |
| Aspect ratio | 1:1, 4:5 | 1:1, 4:5, 16:9, original |



## Automatic sharing to Facebook
Automatic sharing to Facebook is available for both posts and stories. However, **Facebook sharing is incompatible with the schedule feature**.
